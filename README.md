## 📌 About The Project

**FinWise** is a comprehensive financial management tool developed to track income and expenses seamlessly. It analyzes user spending patterns to provide actionable insights, helping users optimize their daily, weekly, and monthly expenses. 

This project is currently under active development as a university graduation project, built with a strong emphasis on scalable software engineering principles and collaborative teamwork.

## ✨ Features

* **Expense & Income Tracking:** Easily log daily financial transactions.
* **Smart Categorization:** Categorize spending (e.g., Groceries, Rent, Salary, Transport).
* **Visual Analytics:** Interactive charts and statistics for daily, weekly, and monthly analysis.
* **Financial Goals:** Set and monitor savings targets (e.g., Car, Travel).
* **Cloud Sync:** Secure data storage and real-time updates.

## 🛠️ Tech Stack & Architecture

This application is built with a focus on writing clean, maintainable, and scalable code.

* **Framework:** [Flutter](https://flutter.dev/) (Dart)
* **Backend & Database:** Firebase (Authentication, Cloud Firestore)
* **State Management:** [BLoC / Cubit](https://bloclibrary.dev/) (Business Logic Component)
* **Architecture:** **Clean Architecture**
  * **Presentation Layer:** UI components, Widgets, and Bloc/Cubit state holders.
  * **Domain Layer:** Business logic, Use Cases, and Entities (Independent of any external libraries).
  * **Data Layer:** Repositories implementations, Data Sources (Remote/Local), and Models.
